I developed this movie streaming web application that provides users with the latest movie updates sourced from The Movie Database (TMDb) using API requests. The app allows users to browse through a collection of movies, view details such as synopsis, release date, and ratings, and watch trailers directly from YouTube. It offers a seamless user experience with intuitive navigation and responsive design.

Technologies Used:

Frontend Framework: React.js
Styling: Emotion (version 11.11.0)
Material-UI (version 5.13.0)
State Management: Redux Toolkit (version 1.9.5),
React-Redux (version 8.0.5)
Routing: React Router DOM (version 6.11.1)
HTTP Requests: Axios (version 1.4.0)
Date Management: Day.js (version 1.11.7)
Form Handling: Formik (version 2.2.9), Yup (version 1.1.1) Utility: Query-string (version 8.1.0)
UI Components: Material-UI Icons (version 5.11.16),
Material-UI Lab (version 5.0.0-alpha.129)
Swiper (version 9.3.2)
Jest-Dom (version 5.16.5)
Notification: React-Toastify (version 9.1.3)

Key Features:

Seamless integration with TMDb API for real-time movie updates.
Dynamic rendering of movie details including synopsis, release date, and ratings.
YouTube trailer links provided for each movie.
Responsive design ensuring optimal viewing experience across devices.
Efficient state management using Redux Toolkit for enhanced performance
Form validation and error handling implemented with Formik and Yup.
User-friendly navigation facilitated by React Router DOM.
Interactive UI components from Material-UI and Swiper.
Notifications for important updates or events via React-Toastify.
