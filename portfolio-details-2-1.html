Welcome to my Sudoku Game! This project is a web-based implementation of the classic Sudoku puzzle game, built using HTML, CSS, and JavaScript. Sudoku is a logic-based number placement game that challenges players to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 subgrids contain all of the digits from 1 to 9.

Features:

Interactive Sudoku board with a clean and intuitive user interface.
Ability to input numbers and solve the puzzle in real-time.
Check for correctness feature to validate user inputs.
Timer functionality to track your solving speed.
Three levels of difficulty: easy, medium, and hard, catering to players of all skill levels.

Technologies Used:

HTML5 for the structure and layout of the web page.
CSS3 for styling the user interface and creating a visually appealing design.
JavaScript for implementing game logic, user interactions, and timer functionality.
Vercel for hosting the live demo of the Sudoku game.
